Variables, expressions, and assignments in Java are similar to their counterparts in most other general purpose languages. We will learn about the details in this section.

Identifiers

Identifier is the name of a variable or other items (class, method, object..etc) defined in a program.

Java Identifier Naming Rules:

  1. must not start with a digit, and all the characters must be letters, digits, or the underscore symbol.
  2. can theoretically be of any length
  3. Java is a case-sensitive language that distinguishes between upper- and lowercase letters in the spelling of identifiers. This means that (Rate, rate, and RATE) are the names of three different variables.

Keywords & Reserved Words

They are a special kind of identifiers that have a predefined meaning in Java.

  • Cannot be used as names for variables or anything else.
  • Examples: public, class, void, static

Some predefined words, like System and println , are not keywords, and you can redefine them. They are defined in libraries required by the Java language standard.

  • you should avoid redefining them because it can be confusing and dangerous, so just treat them as keywords as a safer practice.

Variables

You must always declare variables before using them.

  • declaring a variable is telling the compiler what data type will be stored in that variable.
  • Examples:
    • int grade;
    • float price, percentage;

Assignment Statements

The simplest way to change a variableâ€™s value is by using an assignment statement. After a variable is declared, you can assign a value to it by using an assignment statement.

  • The equal sign = is used as the assignment operator.
  • variable = expression;
  • An assignment statement is an expression that evaluates a value, which is assigned to the variable on the left side of the assignment operator.
  • an assignment expression only evaluates a value, and does not take into account the variable.

An expression represents a computation involving values, variables, and operators that, when taking them together, evaluates to a value.

  • Example:
// Assign 1 to variable y int y = 1; // Assign 1.0 to variable radius double radius = 1.0; // Assign the value of the expression to x int x = 5 * (3/2); // Assign the addition of y and 1 to x x = y + 1;

The assignment operator is executed from right to left. Java also has the facility of chain assignment operators, where we can specify a single value for multiple variables.

var1 = var2 = 10; //both var1 and var2 have the value of 10

Initializing Variables

Initialized variables: the variables that are declared and given a value.
Uniitialized variables: the variables that are declared without a value.

int var1 = 10; //initialized int var2; //uninitialized //we can have both in the same declaration int firstName = John, lastName;

Shorthand Assignment Statements

We can combine the assignment operator (=) and an arithmetic operator so that a given variable can have its value changed by adding, subtracting, multiplying, or dividing by a specified value.

  • The general form is: Variable Op = Expression
  • Which is equivalent to: Variable = Variable Op (Expression)

Op: can be any of +. -. *, / , %
