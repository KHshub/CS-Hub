3.1 â€“ If Statement & Nested If

You can use these conditions to perform different actions for different decisions.
Java has the following conditional statements:

  1. if a specified condition is true, use if to specify a block of code to be executed,

    2. if (20 > 18) { System.out.println("20 is greater than 18"); }

    Output:
    20 is greater than 18

  2. if the same condition is false, use else to specify a block of code to be executed

    3. int time = 20; if (time < 18) { System.out.println("Good day."); } else { System.out.println("Good evening."); }

    Output:
    Good evening.

  3. if the first condition is false, use else if to specify a new condition to test

    4. int time = 22; if (time < 10) { System.out.println("Good morning."); } else if (time < 20) { System.out.println("Good day."); } else { System.out.println("Good evening."); }

    Output:
    Good evening.

  4. Use switch to specify many alternative blocks of code to be executed (we will have more info about it in the next sec )
Nested if means if-within-if.
  • Example on nested-if :
a=10; int b=20; if(a==10){ if(b!=20){ System.out.println("Hello"); } else{ System.out.println("Hi"); } }

Output:
Hi
