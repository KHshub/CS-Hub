5.1 â€“ Static Methods and Static Variables

The static keyword in Java is used to share the same variable or method of a given class.

  • Static Variables:

    • A single copy of the variable is made and shared by all objects at the class level when a variable is specified as static. Essentially, static variables are global variables. The same static variable is shared by all class instances.

      class A { static int b; }
  • Static Methods:

    • A method is said to as static when the static keyword is used in its declaration. The main() function is the most typical illustration of a static method. As was mentioned previously, any static member can be accessed without referencing any other objects and before any objects of its class are generated.

      Static void A() { }

When to use static variables and methods?
Use the static variable for the property that is common to all objects. Use static methods for changing static variables.
