1.3 – The Class String

In Java, there is no primitive type for strings, but there is a class called String which is used to store and process strings. The string value is enclosed with quotes, and the value is usually called an object of type String

  • Example:
//a variable of type String: String greeting = "Hello world!";

Concatenation of Strings

You can join to strings by using + operator, this process is called concatenation.

  • Example (1):
//a variable of type String: String greeting = "Hello world!";
  • Example (2): you can add numbers to strings
String age = “My age is ”; String sentence = age + 22; System.out.println(sentence); //output: My age is 22

Classes, Objects, and Methods

A class is the name for a type whose values are objects.

The class is like a template that describes the object.


Objects store data and take actions which are called (methods).

Methods can return a value or perform an action.

Syntax for calling a method: objectName.methodName()

  • all objects under the same class have the same methods, but they can have different data values.
  • You can include data (here called arguments) in the parenthesis only if needed by the method.
  • Example:
String greeting = "Hello"; int n = greeting.length(); //returns 5
