There are three different types of inheritence:

  1. Single
  2. MultiLevel
  3. Hierarchical



Syntax:

1) When a class inherits another class:

// Single Inheritence: class A { ... } class B extends A { ... }

2) When there is a chain of inheritance:

// Multilevel Inheritence: class A { ... } class B extends class A { ... } class C extends class B { ... }

3) When two or more classes inherit a single class:

// Hierarchical Inheritence: class A { ... } class B extends class A { ... } class C extends class A { ... }
