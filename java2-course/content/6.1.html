6.1 Intro to Swing


Java Swing is a component of Java Foundation Classes. It is used in producing lightweight desktop applications since it can be used to generate window-based programs.

The Java AWT (Abstract Window Toolkit) Package is the original Java package for doing GUIs.

The Swing package is an improved version of the AWT

A GUI is a graphical user interface, it's a windowing system that interacts with the user.

To create graphical user interfaces with Java, two different libraries are used: Java Swing and Java Abstract Window Toolkit (AWT).

However, they differ in ways that, depending on what a person wants to do, would make one more acceptable than the other.

The Differences Between AWT and Swing:

AWT Swing
Heavyweight components Lightweight components
Doesn't support pluggable look and feel Suppports pluggable look and feel
Programs are not portable Programs are portable
An old framework for creating GUIs A new framework for creating GUIs
Components require java.awt package Components require javax.swing package
Supports limited number of GUI controls Provides advances GUI controls like Jtable, JTabbedPane, etc...
More code is needed to implement AWT controls functionally Less code is needed to implement Swing controls functionally
Doesn't follow MVC Follows MVC
