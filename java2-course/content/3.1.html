Polymorphism is the ability of an object to assume various forms.

There are two types of polymorphism:

  1. Compile Time Polymorphism: Also known as Static Polymorphism. Furthermore, the call to the method is resolved at compile-time. Compile Time Polymorphism is achieved through Method Overloading.

  2. Runtime polymorphism: Also popularly known as Dynamic Binding or Dynamic Method Dispatch. In this process, the call to an overridden method is resolved dynamically at runtime rather than at compile-time. You can achieve Runtime Polymorphism via Method Overriding.

Method Overloading Method Overriding
Method overloading is a compile time polymorphism Method overriding is a run time polymorphism
Occurs within the class Performed in two classes with inheritence
May or may not require inheritence Always needs inheritence
Methods have the same name but different signatures Methods have the same name and signature
