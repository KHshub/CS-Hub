6.3 - Applying Swing


To use Swing components, first, you need to import the Swing package:

import javax.swing.*;
We could import the components we'll be using individually, or we could use *. As Shown in the example above, it imports the whole package.

After importing the swing package we can start creating And then adding components! First of all, we need a container, a JFrame. How do we create one? Like this:

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); } }

How about we try creating a JButton next:

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); } }

We can also set the bounds of the button, notice the order of variables, is x axis, y axis, width, height.

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); b.setBounds(130,100,100, 40); //x axis, y axis, width, height } }

After we've created and set the size of our button "b", we can now go ahead and add it to our frame "f".

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); b.setBounds(130,100,100, 40); //x axis, y axis, width, height //add button b to our JFrame f f.add(b); } }

Since we've added our button, we can go ahead and set its size, note that the order here is width, then height (w,h):

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); b.setBounds(130,100,100, 40); //x axis, y axis, width, height //add button b to our JFrame f f.add(b); //400 width and 500 height f.setSize(400,500); } }

We won't be adding a layout manager so we can go ahead and set the value of it on our frame to null.

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); b.setBounds(130,100,100, 40); //x axis, y axis, width, height //add button b to our JFrame f f.add(b); //400 width and 500 height f.setSize(400,500); //Not using layout managers f.setLayout(null); } }

And finally, we need to set the visibility of the frame to true!

import javax.swing.*; public class FirstSwingExample { public static void main(String[] args) { //create an instance of JFrame JFrame f=new JFrame(); //create an instance of JButton JButton b=new JButton("click"); b.setBounds(130,100,100, 40); //x axis, y axis, width, height //add button b to our JFrame f f.add(b); //400 width and 500 height f.setSize(400,500); //not using layout managers f.setLayout(null); //making the frame visible f.setVisible(true); } }

You've learned how to create a frame, add a button, set its bounds, set its size, and make the frame visible.
