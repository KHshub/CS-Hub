1.1 â€“ What is Inheritance?

Inheritance is a mechanism in which one object acquires all the properties and behaviors of a parent object.

It is an essential part of the Object-Oriented programming paradigm.

The concept of inheritance in Java is that new classes can be constructed on top of older ones. You can use the parent class's methods and properties when you inherit from an existing class. You can also add additional fields and methods to your existing class.

You might ask why is inheritance used? Well, there are two main reasons:

  1. To reuse already existent code
  2. For method overriding.

To explain, Method Overriding is when methods have the same name but different signatures, it allows a child class to provide a specific implementation of a method that is already provided by one of its parent classes.

An Example of Method Overriding:

An overriden method called from the child class.



// Parent Class class Parent { void show() { System.out.println("Hello parent class!"); } } // Child class class Child extends Parent { // This method overrides show() of Parent @Override void show() { super.show(); System.out.println("Hello child class!"); } } // Driver class class Main { public static void main(String[] args) { Parent obj = new Child(); obj.show(); } }
OUTPUT:

hello parent class!
hello child class!

The following are some terms you will see a lot when it comes to inheritance:

Sub Class/Child Class: A class that inherits the other class. It is also called a derived class, extended class, or child class.

Super Class/Parent Class: The class from which a subclass inherits its features. It is also referred to as a base class or a parent class.
