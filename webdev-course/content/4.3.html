4.3-Types of styling


  1. Inline : the style specifications are placed within the HTML elements.
    • Uses the style attribute of the HTML element.

    Should be avoided in two cases:

    • If we want to apply the same style declaration to different elements every time.
    • An Inline style sheet mixes the content with the presentation.
      So, if you want to avoid this mixing up, don't use the Inline style sheet.
  2. Embedded or internal: The style specifications are defined within the head section of the web page or HTML page.
    • The alternate method of attaching a style sheet to
      HTML document is < style > tag.
  3. External:
    • The external CSS file contains only CSS code & it is saved with a “.css” extension.(files are kept separately from an HTML document.)
    • The CSS file is used as an external style sheet file in HTML
      document by using a <link> instead of < style >
    • The <link> is placed in the <head> section of the HTML document.

    Attributes of <link> tag:

    1. rel : specify a relationship of CSS with HTML document.
      • Its default relationship value is “style sheet”.
      • Possible relationship values are stylesheet/alternate
        stylesheet.
    2. Type : specifies which type of style language is used.
      • Is not used in The META tag.
      • The value of the type attribute is “text/CSS”.
    3. href : It specifies the path of the style sheet file which is linked
      with the HTML document.
      • points to the external style sheet file's URL.
