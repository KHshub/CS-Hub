3.1-Table


Web developers can organize data into rows and columns using tables.

Table tags :




  • cellpadding: is spacing inside the columns or cells , create with &
  • cellspacing: is spacing outside the columns or cells(space between cells)

Example of table:

<!DOCTYPE html>
<html>
<style>
table, th, td {
border:1px solid black;
}
</style>
<body>
<h2>A basic HTML table</h2>
<table style="width:100%">
<tr>
<th>student</th>
<th>id</th>
<th>grade</th>
</tr>
<tr>
<td>sara</td>
<td>123</td>
<td>3.4</td>
</tr>
<tr>
<td>alaa</td>
<td>234</td>
<td>3.1</td>
</tr>
</table>
</body>
</html>
