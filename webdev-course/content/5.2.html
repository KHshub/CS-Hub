5.2-Manipulating positions


  • A “position” element is one whose position is anything except static.
  • The values for position property are:
  1. position: static
    • HTML elements are positioned static by default.
    • Static position elements are not affected by the top, bottom,
      left,and right properties.
    • It is always positioned according to the normal flow of the page.
  2. position: relative
    • It changes the position from its normal position.
    • Set using: top, right, bottom,and left properties.

    Example of relative:

    HTML file:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Playing with Positioning</title>
    </head>
    <body>
    <div class="b1"> </div>
    <div class="b2"> </div>
    <div class="b3"> </div>
    </body>
    </html>

    CSS file:

    body {
    }
    .b1 {
    background-color: #EEEDDE;
    position: relative;
    height: 200px;
    width: 200px;
    }
    .b2 {
    background-color: #E0DDAA;
    position: relative;
    left:200px;
    bottom:200px;
    height: 200px;
    width: 200px;
    }
    .b3 {
    background-color: #203239;
    position: relative;
    left:400px;
    bottom:400px;
    height: 200px;
    width: 200px;
    }

  1. position: fixed
    • It fixes the position of the element to the same place even if the
      page is scrolled.
    • Set using: top, right, bottom , left properties.
    • It does not leave a gap in the page where the page is normally
      located.
  2. position: absolute
    • It defines the pixel value where the specified HTML will appear.
    • The point of origin is the top-left of the browser.

    Example of absolute:

    HTML file:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Playing with Positioning</title>
    </head>
    <body>
    <div class="b1"></div>
    <div class="b2"></div>
    <div class="b3"></div>
    </body>
    </html>

    CSS file:
