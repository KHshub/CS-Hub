4.2-CSS Rules


  • A CSS consists of a selector & declaration block.
  • Rule = selector + declaration.
  1. Declaration : It defines the property & value of the selector. Example - color : yellow.
    • Property : It is assigned to the selector for manipulating its style.
      Example: background, border, text-align
    • Value : It is assigned to set a property.
      Example : the property color value can be set to yellow
  2. Selector : It is the name of an HTML element.
    • it is applicable to a declaration.

    Types of selectors :

    1. The Element / Tag selector : It is used to define tags associated with HTML.

      2. Syntax: TagSelector {Property : Value; } , Example : H1, p {font-family : times roman;}


    2. Class selectors : It is used to define styles without redefining plain HTML tags.

      3. Syntax: .ClassSelector { Property : Value; }, Examples: (next slide)


    3. ID selectors: used to define styles relating to objects with a unique ID.

      4. Syntax: #IDSelector {Property : value;} Examples: (next slide)


    4. Grouped Selectors : If there are elements with the same style definitions, then it is easy for the group selector to minimize the code.
      5. Group selectors are specified with comma (,) in between their names.

      Syntax: Selector1, selector 2, ....{property value;}


    5. Context Selectors VI.CSS comments : Comments are denoted within a style sheet like in C programming (//).

      6. They do not work in the HTML code.
