2.4-Memory allocation and Errors

  • Definition statement: A declaration that defines how much memory is needed for data storage,and what you need when you execute a program.

  • Three items associated with each variable:
    • Data type
    • Actual value stored in the variable (its contents)
    • The memory address of the variable

  • Address operator (&) provides the variableâ€™s address Memory Allocation

    • #include <iostream>
    using namespace std;
    int main()
    {
    int age = 50;

    cout << age << endl; //shows value that is stored in age
    cout << &age << endl; //shows address of the memory location

    return 0;
    }

    Output

    50 //age output
    0x23fefc //age address memory location
  • When a variable is declared, memory is allocated according to the data type.
    • You can declare a varible Multiple times put you can define it once

    Determining Storage Size

    Floating-Point Types Floating-point number (real number):Zero or any positive or negative number containing a decimal point.

  • Three floating-point data types in C++: float (single precision) , double (double precision) , long double
  • float literal: Append an f or F to the number
  • long double literal: Append an l or L to the number


    • #include <iostream>
    using namespace std;
    int main()
    {
    cout << 0.3f<< endl; // float literal
    cout << 1/3.f<< endl; // float literal
    cout << endl; // break line

    cout << 0.3 << endl; // double literal
    cout << endl; // break line

    cout << 0.3L << endl; // long double literal
    cout << 1/3.L << endl; // long double literal
    cout << endl; // break line

    cout << 1/3 << endl; // integer
    }

    Output

    0.3
    0.333333

    0.3

    0.3
    0.333333

    0

  • A unique feature of C++ is that you can see where and how values are stored
  • sizeof() operator provides number of bytes used to store values of the data type named in the parenthesis
    • Values returned by sizeof() are compiler dependent
  • No special characters are allowed

#include <iostream>
using namespace std;
int main()
{
int a;
float b;
double c;
char d;
int e = 10;
cout << "sizeof(int): " << sizeof(a) << endl;
cout << "sizeof(float): " << sizeof(b) << endl;
cout << "sizeof(double): " << sizeof(c) << endl;
cout << "sizeof(char): " << sizeof(d) << endl;

cout << sizeof(e) << endl; //number of bytes not the value
}

Output

sizeof(int): 4 sizeof(float): 4 sizeof(double): 8 sizeof(char): 1 4

Common Programming Errors

  • Omitting the parentheses after main()
  • Omitting or incorrectly typing the opening brace { or the closing brace } that signifies the start and end of a function body.
  • Misspelling the name of an object or function.
  • Forgetting to enclose a string sent to cout with quotation marks.
  • Omitting a semicolon at end of statement.
  • Adding a semicolon at end of #include statement.
  • Missing \n to indicate new line.
  • Substituting letter O for zero and vice versa.
  • Failing to declare all variables.
  • Storing an incorrect data type into a variable.
  • Attempting to use a variable with no value.
  • Dividing integer values incorrectly.
  • Mixing data types in the same expression.
