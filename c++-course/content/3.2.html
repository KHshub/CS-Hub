3.2-Program Input Using cin

  • cin Object:Allows data entry to a running program
    • When using the cin object, the program waits for keyboard input.
    • The software resumes running after the keyboard entry is complete, using the data that was entered.
  • An output statement preceding the cin object statement provides a prompt to the user
  • cin can accept multiple input values to be stored in different variables
  • Multiple numeric input values must be separated by spaces (cin >> num1 >> num2)


#include < iostream >
using namespace std;
int main( ) {
int age;
cout << "Enter your age: ";
cin >> age;
cout << "Your age is: " << age << endl;
}
