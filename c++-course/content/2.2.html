2.2-Data types and Declaration Statements

Data type: A set of values & the operations that can be applied to these values.

  • Two fundamental C++ data groupings:
    • Class data type (a class): Created by the programmer.
    • Built-in data type (primitive type): Part of the C++ compiler.

    Literal (constant): An actual value.

    Integer: A whole number, C++ has nine built-in integer data types:

    1. int data type: Whole numbers (integers), optionally with plus (+) or minus (–) sign , Example: 2,-5
    2. char data type: Individual character; any letter, digit, or special character enclosed in single quotes , Example: ‘A’ , '1'
    3. bool data type: Represents Boolean (logical) data
      Restricted to two values: true or false
    4. Signed & Unsigned Data Types
      • Signed data type: Allows zero, positive, and negative values
      • Unsigned data type: Permits only positive and zero values ( provides double the range of its signed counterpart)



    Variables & Declaration Statements

    Variable: Is an identifier for a memory address where data can be held.

  • Declaration statement: Specifies the data type and identifier of a variable; sets up the memory location,and what you need when using the variable.
    • A single declaration statement can define several variables of the same data type.
    • Variables can be initialized in a declaration.
    • Before being utilized, a variable must be declared.

      • Example :

      #include <iostream>
      using namespace std;
      int main()
      {
      //declare statement
      // datatype variable_name ;
            int   var1;
          float   var2;
          double   var3;
          char   first;
          bool   choice;
      }


  • Assignment statement ( initialization ): Used to store a value into a variable.
    • The memory address of the variable on the left side is allocated the value of the expression on the right.

      • Example :

      #include <iostream>
      using namespace std;
      int main()
      {
      //declare statement
      // datatype variable_name ;
           int   var1 = 2;
          float   var2 = 2.3;
          double   var3 = 3.523;
          char   first = 'a';
          bool   choice = true;

      //you can do it like this
           int var4;
           var4 = -2;


      //To print the variable

      //   variable   end_line
      cout<<   choice <<   endl ;

      //you can do it like this also
      cout << var1 << endl << var4 << endl ;

      }
