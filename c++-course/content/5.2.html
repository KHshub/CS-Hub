5.2-Arrays


One-dimensional arrays

  • A collection of connected items of the same data type that are kept under one group name (array name).
  • Syntax


    • dataType arrayName[number of elements] ;

  • Element: A component of the array (Array storage of elements is contiguous).
  • Index (subscript) of an element: The position of the element within the array (Indexes are zero-relative ,Strat from zero)
    • To reference an element, use the array name & the index of the element
  • Out of range array Indexes are not validated during compilation.
    • May produce run-time errors
    • May replace a value in the mentioned memory region and result in additional problems.


    Array initialization

  • The array declaration statement allows for the initialization of array items.
  • Initialization:
    • Can span multiple lines, because white space is ignored in C++.
    • In the absence of sufficient values, element 0 of the array is used as the first element.
  • It is possible to omit the size when initializing in the declaration.
  • At the end of the string, the Char array will include one additional null character.

    • Example:

    int numbers[] = {2 , 3 ,4 , 6};

    Declaring & processing 2-D arrays

    Syntax

    dataType arrayName[rows][columns] ;

  • The array declaration must specify both dimensions (row is specified first, then column).
  • Any time an array element is referenced, both dimensions must be given.
  • Nested for loops are employed to handle two-dimensional arrays.
    • The outer loop controls the rows
    • The inner loop controls the columns


    Arrays as arguments

  • When sending an array to a function, use the array's name as the argument.
  • Arrays passed as arguments are passed by reference.
  • Passing by value is used for individual array elements as arguments.
