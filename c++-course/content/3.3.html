3.3-Mathematical Library & Type conversion

Using Mathematical Library Functions

  • You must include the cmath header file: #include <cmath>
  • Math functions require one or more arguments as input, but will return only one value
  • All functions are overloaded, & can be used with integer and real arguments




#include <iostream>
#include <cmath>

using namespace std ;

int main(){
float x; cout << "Enter x: ";
cin >> x ;
cout << "The log (" << x << ") is " << log(x) << endl;
return 0;
}

Lets pretend i entered 3

Enter x: 3
The log (3) is 1.09861


Type conversion

  1. Cast operatorA unary operator that forces the data to the desired data type. Cast does not need to include cmath library.(Example : int(a+b)= int(2.3+3.6)=int(5.9) =5)
  2. Coercion:Forcing a data value to another data type
    • When evaluating an assignment statement, the value of the expression on the right side will be coerced (converted) to the data type of the variable on the left side.
    • An assignment statement's right side may also contain a variable from the left.
