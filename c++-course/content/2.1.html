2.1-Structure of C++


Modular program: An application made up of connected sections (or modules) that are organized logically and understandably.
Advantages: Easy to develop, correct,and modify.


    Modules in C++can be classes or functions
  1. Function(Method): Accepts an input, processes the input, & produces an output.

    • The processing of a function is encapsulated and concealed within the function.

    • Rules of Function Names
      • Require a set of parentheses at the end ()
      • Can use mixed upper and lower case Z or z
      • Should be meaningful, or be a mnemonic (you can remember it)

        • Examples: cricle_Area() , _factorial()

      Function header line: beginning of a function, it includes:

    • The kind of information that is returned by the function (if any)
    • The name of the function
    • The type of data that must be passed into the function when it is invoked (if any)

      • Example: float AreaTriangle (float base,float height)

    Arguments: The data passed into a function.The argument is float base,float height

    Function body: The braced statements within a function { } .

    float AreaTriangle(float base,float height){this is the function body;}

    Any statement inside the body must end / terminated with semicolon ;

    The main() function

    Each C++ program has at least one function, the main() function or the driver function
    All other functions are invoked from main()

    return: A term that instructs the function to return the proper value.

    The statement return 0; in the main() function causes the program to end
  2. Class: Includes bothÂ data as well as functions for manipulating the data.

    3. Preprocessor command: Starts with a # causes action before the source code is compiled into machine code. (Executed before compilation and runtime)

    Header files: Files included at the head (top) of a C++ program.

    #include <file name> : Causes the named file to be inserted into the source code.

    Using namespace: Indicates where the header file is located.

    Namespaces qualify a name


    cout object: A data output object that transmits information to a common output display device.

    Identifier: A name given to an element of the language, such as a variable, constant, class or function.

    • C++ is a case-sensitive language which means int a doesn't mean int A
        • Rules for forming identifier names:
      • The first character must be a letter A / a or underscore _
      • Only letters, numbers, or underscores are permitted after the first character (no blanks allowed)
      • The maximum length of an identifier = 1024 characters
      • Keywords cannot be used as identifiers.

      Valid C++ Identifier : addNums , degToRad , findMax , density , slope.

    Keyword(reserved): A reserved name that represents a built-in object or function of the language.





String: Any grouping of letters, numbers, and special characters encapsulated in "double quotes" .

Delimiter: A symbol that marks the beginning and end of a string; not part of the string .

Escape sequence: One or more characters preceded by a backslash \ (like \n) ,(\) Tells the compiler to treat the following characters as special instruction codes.
