1.1-Introduction Computer Languages


Definitions:


Computer program(software): a set of instructions used to operate a computer to produce a specific result.

Program: solution developed to solve a particular problem, written in a form that can be executed on a computer.

Programming: Process of writing a software .

programming languages? A set of instructions, which are used to build programs, can take many different shapes and forms.

Types of programming languages:




Programs can also be classified by their orientation:


Procedural: Available instructions are utilized to develop procedures, which are self-contained units.

Object-oriented: The manipulation of reusable objects that contain code and data.

Source code: The programs written in a high- or low level language by programmers.

Program Handling: The source code is first Compiled or interpreted. Then if it is contained in more than one file, the files are linked, then the execution is performed.


How does code get translated?

is accomplished by converting source code into machine instructions.
using Two methods:

  1. Interpreter: Every sentence is translated individually and executed right after translation.

  2. Compiler: All statements are translated, stored, and later executed as executable programs or object programs.


  • Linker: all of the compiled code required for the program.
  • Assemblers: Translate programs into machine language.
