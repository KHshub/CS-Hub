3.1-Formatting Numbers for Program Output

cout cout with stream manipulators can control output formatting

    Formatting floating-point numbers requires three field-width manipulators to:

  1. Set the total width of the display (setw)
    • The setw manipulator setting is disregarded if the field width is too short, and sufficient space is allocated for printing.

  2. Force a decimal place (fixed)

  3. Set the number of significant digits after the decimal point (setprecision)
    • setprecision: Sets the total number of shown digits unless a decimal point has been expressly required, in which case it sets the number of digits following the decimal point.
    • The fractional part of the value is rounded to the provided number of decimal places if the setprecision parameter is too small.
    • If setprecision value is excessively huge, and the fractional value is shown in its current size.
