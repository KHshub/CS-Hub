4.1-Selection


Selection statements (conditional statements) code (statements) that is executed only when a certain condition is satisfied.

Could be:

  • if statment
  • switch statment

if statement

if-else statement: implements a decision-making process for two options.


Syntax
#include <iostream>
#include <cmath>

using namespace std ;

int main(){
if (condition)
{ statements executed if condition is true;}
else
{statements executed if condition is false; }
}


    A numerical assessment of the condition is made:
  • A zero is false
  • A non-zero is true
The else is optional Executed only if (if statement) is false.

if statement Example:



Relational expression: Uses relational operators to compare two operands or expressions.

are only assessed to a value of 1 or 0:


  • If the value is 1, it is true
  • If the value is 0, it is false

For comparison purposes, char values are automatically converted to int values.


AND (&&): Expression one and two are true only then the output is true.


OR (||): If any one or both of the expressions is true, the condition is true.


NOT (!): Transforms a statement such that it has the opposite meaning—true becomes false, false becomes true.


If-else executes commands dependent on the outcome of a comparison.

Compound statement: A series of short phrases enclosed in braces.


Scope: Based on the location where the variable was declared


A variable’s scope: is based on where the variable is declared


One-way selection: if not followed by the optional else clause.

Nested if statement: An if-else statement inside another if-else.


if-else chain: A nested if statement that appears in the else section of an if-else statement.




switch statement

switch statement: allows for one choice from a variety of options, Starts the statement with the switch keyword


default case: Is executed if no other case were found default case is not required.


case keyword : Chooses a value to be compared to the switch expression.


we use "break" with switch

Data validation: Verify user input using defensive programming approaches.


Solving quadratic equations: The process for developing software can be used to find the
roots of a quadratic problem.


Defining the structure using stacked if statements without brackets.

switch expression: Assessed and measured against each case value.


Example on switch


#include <iostream>
#include <cmath>

using namespace std ;

int main(){
int num = 2; switch (num) { case 1: cout << "ahmed"; break; case 2: cout << "khaled"; break; } // output "khaled" because of (num)
